## 🛠 提交规范 (Commit Convention)

`类型: 描述 (Type: Description)`

| 类型 | 标识符 | 描述 |
| :--- | :--- | :--- |
| **功能** | `feat` | 编写了新的算法模型（如：Linear Regression） |
| **文档** | `docs` | 学习笔记、Jupyter Notebook 或 README 修改 |
| **修复** | `fix` | 修复了代码 Bug 或公式推导错误 |
| **重构** | `refactor` | 优化代码结构或矩阵运算性能 |
| **配置** | `chore` | 修改 `.gitignore` 或 IDE 环境配置 |
